# Commands to Run

Follow these commands in order to set up and run your Forex Trade Journal app:

## 1. Navigate to the project directory
```bash
cd "c:\Users\ILYAAS ABDIRAHMAN\Desktop\journal"
```

## 2. Install dependencies
```bash
npm install
```

## 3. Start the development server
```bash
npm run dev
```

The app will open automatically in your browser at `http://localhost:3000`

---

## Additional Commands

### Build for production
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

## Features Included

✅ **Complete Trade Logging** - Log all 25 data points per trade
✅ **Pre-Trade Checklist** - Customizable checklist before trade entry
✅ **Dashboard** - Key metrics at a glance
✅ **Analytics** - Charts and insights (Equity curve, Win/Loss, Strategy performance, etc.)
✅ **Trade History** - Search, filter, and review all trades
✅ **Settings** - Account settings, checklist editor, data export/import
✅ **Responsive Design** - Works on desktop, tablet, and mobile
✅ **Premium UI** - Modern glassmorphism design with smooth animations
✅ **Data Persistence** - All data saved to localStorage
